class Markov\+::\+Edge$<$ Node\+Storage\+Type $>$}

\mbox{\hyperlink{class_markov_1_1_edge}{Edge}} class used to link nodes in the model together. 

Has left, right, and weight of the edge. Edges are {\itshape UNIDIRECTIONAL} in this model. They can only be traversed left to right.

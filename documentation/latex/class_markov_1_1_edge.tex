class Markov\+::\+Edge$<$ Node\+Storage\+Type $>$}

\mbox{\hyperlink{class_markov_1_1_edge}{Edge}} class used to link nodes in the model together. 

Has Left\+Node, Right\+Node, and Edge\+Weight of the edge. Edges are {\itshape U\+N\+I\+D\+I\+R\+E\+C\+T\+I\+O\+N\+AL} in this model. They can only be traversed Left\+Node to Right\+Node.

\doxysubsection{Detailed Description}
An abstract class for \mbox{\hyperlink{namespace_markov_1_1_random}{Random}} Engine. 

This class is used for generating random numbers, which are used for random walking on the graph.

Main reason behind allowing different random engines is that some use cases may favor performance, while some favor good random.

\mbox{\hyperlink{class_markov_1_1_random_1_1_mersenne}{Mersenne}} can be used for truer random, while \mbox{\hyperlink{class_markov_1_1_random_1_1_marsaglia}{Marsaglia}} can be used for deterministic but fast random.

Markov::Node* Markov::Node::RandomNext()

Chose a random node from the list of edges, with regards to its weight, and traverse to that.

Returns Node that was chosen at weight biased random.

This operation is done by generating a random number in range of 0-this.total_edge_weights, and then iterating over the list of edges. At each step, weight of the edge is subtracted from the random number, and once it is 0, next node is selected.

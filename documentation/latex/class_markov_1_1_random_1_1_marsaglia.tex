\doxysubsection{Detailed Description}
Implementation of \mbox{\hyperlink{class_markov_1_1_random_1_1_marsaglia}{Marsaglia}} \mbox{\hyperlink{namespace_markov_1_1_random}{Random}} Engine. 

This is an implementation of \mbox{\hyperlink{class_markov_1_1_random_1_1_marsaglia}{Marsaglia}} \mbox{\hyperlink{namespace_markov_1_1_random}{Random}} engine, which for most use cases is a better fit than other solutions. Very simple mathematical formula to generate pseudorandom integer, so its crazy fast.

This implementation of the \mbox{\hyperlink{class_markov_1_1_random_1_1_marsaglia}{Marsaglia}} Engine is seeded by \mbox{\hyperlink{random_8h}{random.\+h}} default random engine. \mbox{\hyperlink{class_markov_1_1_random_1_1_random_engine}{Random\+Engine}} is only seeded once so its not a performance issue.
